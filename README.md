# SQL Notes

A hand-written, growing reference for SQL — built in public.

Two goals: **(1)** anyone should be able to learn SQL from this repo for free, no course required, and **(2)** future-me should be able to reopen any note and get the skill back from scratch.

Every concept here is written in my own words, with real examples I ran myself. If it's in this repo, I understand it — not just copied it.

## How each note is structured

Every note follows the same four-part shape, so it reads like documentation, not scratch paper:

1. **What it is** — the concept in plain English
2. **Why / when to use it** — when a real analyst reaches for it
3. **Syntax + example** — a query I actually ran, with its result
4. **In my own words** — the mental model, explained like I'm teaching a friend

See [`TEMPLATE.md`](./TEMPLATE.md) for the blank template.
